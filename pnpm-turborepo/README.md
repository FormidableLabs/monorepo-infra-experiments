pnpm + turborepo
================

Experiment with pnpm + Turborepo

## Development

Install:

```sh
$ npm install -g pnpm
$ pnpm install
```

Build:

```sh
# Run build across all packages.
$ pnpm run build

# Watch build in all packages
$ pnpm run build:watch
```

See some doggo info:

```sh
$ pnpm -r exec node lib/index.js
```
